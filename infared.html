🛡️ mod commands restricted to mod roles only

🔨 /ban <member> [reason] [duration] [delete_messages]

Ban a member permanently or for a set duration (e.g. 1h, 7d). Auto-unbans when the timer expires.

✅ /unban <user_id> [reason]

Unban a user by their Discord ID.

👢 /kick <member> [reason]

Kick a member from the server.

⏱️ /timeout <member> <duration> [reason]

Temporarily mute a member for up to 28 days (e.g. 10m, 2h, 1d).

⚠️ /warn <member> [reason]

Issue a warning to a member. They'll be notified via DM.

🐢 /slowmode <duration>

Set slowmode on the current channel. Use 0s to disable.

📋 /cases view <member>

View all logged warnings for a member.

🗑️ /cases clear <member>

Clear all warnings for a member.

🔒 thread closing

Add the close tag to any forum thread to trigger the close flow — the bot will ask for a reason, notify the thread owner, then archive and lock it.

📖 bot commands available to everyone

🏠 /serverinfo

See stats about the server like member count, boost level, and creation date.

⭐ /rank [@member]

Check your XP and current level. You earn XP just by sending messages!

📊 /poll <question> [options]

Create a quick Yes/No poll, or pass comma-separated options for a multi-choice vote.

🎫 /ticket open [reason]

Open a private support ticket with the mod team.

🔒 /ticket close

Close your support ticket once your issue is resolved.
